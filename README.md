# Game Studio Tool — LiveOps Desk

Dense LiveOps desk for build triage and playtest feedback. Built as a Design Engineering portfolio case (Next.js + TypeScript + accessibility-minded UI).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build status

- **Step 1 (done):** app shell — tokens, sidebar, Feedback Inbox placeholder
- **Step 2 (done):** mock feedback data, dense table, URL-driven filters
- Step 3+: detail drawer, a11y, stub pages, before/after case
